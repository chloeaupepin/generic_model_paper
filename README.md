# Generic_model_paper

## Overview
Vaccines are a new potential tool to act against antimicrobial resistance (AMR). Here we construct a generic model paper to investigate the impacts of vaccines on AMR. 

## Repo contents 
- [scripts](./scripts): R codes to generate all the analyses in the manuscript and supplementary
- [files](./files): files used in analyses, generated by R codes in scripts folder
- [figures](./figures): figures in the manuscript and supplementary

## Installation guide
Users should install the following packages, from an `R` terminal:

```
install.packages(c("dplyr", "here", "purrr", "tidyr", "deSolve", "furrr", "stringr", "ggplot2", "ggh4x", "scales","lhs","patchwork", "ggrepel", "parallel", "epiR"))
```
